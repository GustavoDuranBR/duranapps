FileWizard

Organize e automatize a organização de arquivos no Windows

📋 Sobre o Aplicativo

O FileWizard é uma ferramenta desenvolvida para facilitar a organização de arquivos em seu computador.

Com ele é possível separar arquivos automaticamente em pastas de acordo com critérios definidos pelo usuário, como extensão do arquivo ou parte do nome.

O objetivo do aplicativo é tornar tarefas repetitivas de organização mais rápidas, eficientes e seguras.

📂 Organização Inteligente de Arquivos

O FileWizard permite organizar arquivos utilizando diferentes critérios de filtragem e agrupamento.

  • Extensão do arquivo: Organize arquivos por tipo (.pdf, .jpg, .txt, etc).
  • Parte do nome: Agrupe arquivos que contenham uma palavra específica no nome.
  • Combinação de critérios: Combine extensão e parte do nome para uma organização ainda mais precisa.

⚙️ Ações de Organização

O usuário pode escolher como deseja organizar os arquivos encontrados.

  • Copy: Copia os arquivos para a pasta de destino mantendo os originais.
  • Move: Move os arquivos para a pasta de destino reorganizando completamente a estrutura.

Essas opções permitem flexibilidade dependendo da necessidade do usuário.

🔍 Filtros de Organização

  • Seleção por tipo de arquivo (.txt, .jpg, .pdf, etc).
  • Filtro por palavras presentes no nome do arquivo.
  • Combinação de filtros para refinar a organização.

Isso permite organizar grandes volumes de arquivos com poucos cliques.

🛠 Tecnologias Utilizadas

  • Python — linguagem principal da aplicação.
  • Tkinter — criação da interface gráfica.
  • Pillow (PIL) — manipulação de imagens.
  • Shutil — operações de cópia e movimentação de arquivos.
  • Datetime — registro de timestamps das operações.
  • OS — manipulação de diretórios e caminhos.
  • ttkthemes — temas visuais adicionais para a interface.
