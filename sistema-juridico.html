Sistema Jurídico

Sistema desktop moderno para gestão de clientes e processos jurídicos

📋 Sobre o Sistema

O Sistema Jurídico foi desenvolvido para auxiliar escritórios de advocacia na organização e gestão de clientes e processos. A aplicação foi projetada para ser simples, rápida e segura, permitindo que advogados e equipes tenham acesso às informações do escritório de forma centralizada.

🔐 Segurança e Gestão de Acessos

A versão atual introduz um sistema completo de controle de acessos, garantindo que apenas usuários autorizados possam visualizar ou alterar informações do sistema.

  • Sistema Multiusuário com login obrigatório.
  • Admin: controle total do sistema e gerenciamento de usuários.
  • Super: cadastro e edição de registros.
  • Padrão: acesso somente para consulta.
  • Painel administrativo para criação e gestão de usuários.

📱 Acesso Mobile

O sistema pode ser acessado diretamente pelo celular ou tablet dentro da mesma rede Wi-Fi do escritório, sem necessidade de instalação nos dispositivos móveis.

  • Detecção automática do IP da máquina na rede.
  • Acesso via navegador em qualquer dispositivo.
  • Interface responsiva adaptada para telas menores.

💾 Backup Inteligente

Para garantir a integridade dos dados do escritório, o sistema possui um módulo de backup inteligente integrado.

  • Seleção de pasta usando o seletor nativo do Windows.
  • Backup automático em pasta segura caso o usuário cancele a seleção.
  • Abertura automática da pasta no Windows Explorer após o backup.

📦 Instalador Inteligente

  • Detecção automática de versões já instaladas.
  • Opção de backup antes da atualização.
  • Preservação automática de usuários, clientes e configurações.
  • Otimização de espaço evitando acúmulo de backups antigos.

📁 Localização dos Dados

Os dados do sistema são armazenados de forma segura em:

%LOCALAPPDATA%\SistemaJuridico\data\clientes
