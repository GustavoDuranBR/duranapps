🚀 Sobre o Duran Apps

O Duran Apps é um projeto dedicado ao desenvolvimento de utilitários e aplicações voltadas para produtividade e automação.

Os aplicativos publicados aqui são criados com o objetivo de facilitar tarefas do dia a dia, melhorar fluxos de trabalho e oferecer ferramentas úteis para profissionais e usuários comuns.

Sistema Jurídico

Um dos principais projetos do Duran Apps é o Sistema Jurídico, uma aplicação desktop desenvolvida para auxiliar escritórios de advocacia na gestão organizada de clientes e informações.

O sistema foi projetado para ser simples de usar e ao mesmo tempo robusto, oferecendo recursos importantes como:

  • Gestão de clientes e registros jurídicos
  • Sistema de login com níveis de acesso
  • Acesso via rede local para dispositivos móveis
  • Módulo de backup inteligente para proteção de dados
  • Interface moderna e responsiva

O objetivo é transformar tarefas administrativas do escritório em processos mais rápidos, organizados e seguros.
